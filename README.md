# 🌸 Bloom & Blossom | Premium Flower Shop Website

A stunningly designed, responsive, and interactive flower shop landing page. Featuring smooth animations, dynamic product sliders, and a functional shopping cart experience.

## ✨ Key Features
- **Interactive Hero Section**: High-quality product showcase with Swiper.js.
- **Dynamic Product Gallery**: Smooth filtering and interactive product cards.
- **Functional Shopping Cart**: Real-time cart updates (add/remove) using modern JavaScript.
- **Micro-Animations**: Premium "Animate On Scroll" (AOS) effects for a professional, lively feel.
- **Integrated Contact & Maps**: Seamless user engagement with built-in location services.
- **100% Responsive Design**: Optimized for Desktop, Tablet, and Mobile devices.

## 🛠️ Tech Stack
- **Languages**: HTML5, CSS3, JavaScript (ES6+)
- **Sliders**: [Swiper.js](https://swiperjs.com/)
- **Animations**: [AOS (Animateon Scroll)](https://michalsnik.github.io/aos/)
- **Icons**: [Font Awesome](https://fontawesome.com/)
- **Fonts**: Google Fonts

## 🚀 Quick Start
1. Clone the repository: `git clone https://github.com/Anjalikasoudhan/Flower-Shop-Website.git`
2. Open `Flower.html` in your favorite browser.
3. Enjoy the blooming experience!
